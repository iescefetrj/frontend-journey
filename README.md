## Sobre

Material didático do canal IES com vídeos e exemplos práticos para aprender HTML, CSS, JavaScript e React. Ideal para quem prefere aprender com projetos e exercícios.

## Ementa das aulas

- **HTML**: estrutura, semântica, formulários e acessibilidade.
- **CSS**: seletores, propriedades, responsividade, Flexbox e Grid.
- **JavaScript**: DOM, eventos, manipulação de elementos, fetch/API e boas práticas.
- **React.js**: componentes, props, estado, hooks básicos e organização de projeto.

## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/frontend-journey.git
cd frontend-journey
```

2. Abra os exercícios e os exemplos no seu editor preferido (VS Code recomendado).
3. Assista às vídeo-aulas e siga os exemplos na pasta correspondente (quando houver).
